<p class="club-lead-intro">The <strong>2023 season</strong> was <a href="/clubs/petite-bourgogne/">Petite-Bourgogne</a>'s first season in their history. In addition to the domestic league, the club also participated in the Coupe du Québec and was given a place in the Canadian Championship.
</div>

On April 5, 2023, Petite-Bourgogne announced that Andy Cook was going to be appointed as the club's first manager.

On April 15, 2023, the club played their first ever competitive game at Stade Oscar Peterson, defeating <a href="/clubs/deux-rives/">Deux-Rives</a> 1-0 thanks to a goal from Ousmane Blot in the 72nd minute.

On October 17, 2023, following a penalty shoot-out after a 1-1 draw, Petite-Bourgogne beat <a href="/clubs/locomotive/">Locomotive</a> to lift the inaugural Coupe du Québec.

    function toComparable(text, isNumeric) {
      if (isNumeric) {
        var num = Number(text.replace(/[^\d.-]/g, ""));

      }

    function sortWithinGroups(columnIndex, direction) {
      var rows = Array.from(tbody.querySelectorAll("tr"));

      var groups = [];
      var current = null;

      rows.forEach(function (row) {
        if (row.classList.contains("squad-group-row")) {
          current = { header: row, items: [] };
          groups.push(current);
        } else if (current) {
          current.items.push(row);
        }

      groups.forEach(function (group) {
        group.items.sort(function (a, b) {
          var aValue = toComparable(getCellText(a, columnIndex), isNumeric);
          var bValue = toComparable(getCellText(b, columnIndex), isNumeric);
          if (aValue < bValue) return direction === "asc" ? -1 : 1;
          if (aValue > bValue) return direction === "asc" ? 1 : -1;
          return 0;
        });

        tbody.appendChild(group.header);
